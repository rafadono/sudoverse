# SudoVerse UI Package (`@sudoku/ui`)

This library is designed to contain shared UI design tokens, assets, and reusable user interface components across the **SudoVerse** clients (Web, Mobile, and Desktop wrappers).

## Purpose

- **Design System Consistency**: Define common styling attributes, layout tokens, and design variables.
- **Component Portability**: Share UI components like buttons, grids, and dialog boxes across multiple client runtimes to avoid code duplication.
- **Tailwind & Native Compatibility**: Provide styling helpers compatible with both web-based Tailwind frameworks and React Native styling subsystems.
